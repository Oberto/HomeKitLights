HomeKit Example

Overview

Provide an example highlighting use of HomeKit. This Application can be used as an example to a prospective client on how HomeKit could be used as part of their IoT project.

Platform

  • iOS 13 and above.

Highlights

Uses some of the newer iOS frameworks including

  1. SwiftUI: New UI framework in iOS 13.
  2. Combine: Reactive Framework new to iOS 13.

Other frameworks

  • HomeKit: Access to the device HomeKit framework
  • os.log: Apple Logging Framework
